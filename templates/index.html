Event Sourced Bank

A sample application using the Python eventsourcing library

This example models a trivially simple bank. It works as follows:

  • Create accounts in the Accounts panel, then submit credit/debit transactions on them
  • In response, the Ledger updates automatically. It shows the bank's overall balance, and the total number of transactions across all accounts
  • The Events shows the underlying events generated. This is the heart of the application; it's what makes it "event sourced"

More info on the eventsourcing library in the docs. Source code for this application is here.
